Privacy Policy

Last Updated: January 2025

1. Information We Collect

YkApp collects and processes the following data through Etsy API:

  • Product listings data (titles, tags, descriptions)
  • Shop information (names, statistics)

2. How We Use Your Information

We use the collected data to:

  • Provide tag analysis and research tools
  • Generate product insights and recommendations
  • Display shop statistics and trends

3. Data Storage

We do not store any personal information. All data is fetched in real-time from Etsy API and processed temporarily.

4. Third-Party Services

We use Etsy API to fetch product data. Please refer to Etsy's Privacy Policy.
